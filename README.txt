# Shifor v1

## Описание
Shifor — это программа для зашифровки и дешифровки текстовых файлов. Она позволяет пользователю безопасно шифровать данные, сохраняя их конфиденциальность.

**Важно!** Не шифруйте текстовые документы дважды. Это может привести к непредсказуемым результатам. Этот баг будет исправлен в будущем, но это не точно :)

## Установка
1. Клонируйте репозиторий:
    ```bash
    git clone <ссылка на ваш репозиторий>
    ```
2. Перейдите в папку проекта:
    ```bash
    cd Shifor
    ```
3. Установите зависимости:
    ```bash
    npm install
    ```
